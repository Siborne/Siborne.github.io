LRU 缓存算法的实现

操作系统，面试笔试常考，基于Java实现的 LRU 缓存算法

2026-01-06

LRU 缓存算法实现笔记（Java）

一、什么是 LRU？

  • LRU（Least Recently Used） 是一种常用的缓存淘汰策略。

  • 核心思想：​当缓存满时，优先淘汰最久未使用的数据。

  • 要求支持：

    • get(key)：获取值，并将该 key 变为“最近使用”。
    • put(key, value)：插入或更新值，并标记为“最近使用”；若容量满，则删除最久未使用的项。

时间复杂度要求：get​ 和 put​ 均为 ​O(1) 。

二、数据结构选择

为了实现 O(1) 的操作，需结合两种结构：

数据结构 作用
HashMap 快速通过key查找对应的节点（O(1)）
双向链表（Doubly Linked List） 维护访问顺序 —— 链表尾部为最近使用，头部为最久未使用

双向链表支持 O(1) 删除中间节点（因为有 last 指针）。

2. 内部类 DoubleList（双向链表封装）

提供三个核心操作：

方法 功能
addNode(node) 将新节点加到​尾部（表示最新使用）
moveNodeToTail(node) 将已有节点移到尾部（被访问/更新时调用）
removeHead() 移除​头部节点（最久未使用），返回该节点

注意边界情况：

  • moveNodeToTail​ 中先判断 if (node == tail) return; 避免空指针。
  • removeHead​ 处理单节点情况（head == tail）。

3. 主类 LRUCache

  • 成员变量：

    • keyNodeMap: HashMap<Integer, DoubleNode>，实现快速查找。
    • nodeList: 自定义双向链表，维护使用顺序。
    • capacity: 缓存最大容量。

  • 构造函数：初始化 map、list 和 capacity。

四、关键细节 & 易错点

  1. 必须保存 ​key​​ 在节点中
    → 否则无法在淘汰时从 HashMap 中删除对应项。

  2. ​moveNodeToTail​​ 必须处理 ​node == tail​​ 的情况
    → 否则在单节点或尾节点被访问时会出错（如空指针）。

  3. 删除头节点后，要同步清理 ​HashMap​
    → 保持 map 和 list 状态一致。

  4. 双向链表的指针操作要小心顺序

    • 断开旧连接（修改前后节点的指针）
    • 再连接到尾部
    • 最后更新 tail

  5. 容量为 0 的边界情况？

    • 题目通常保证 capacity > 0，但实际工程中可加校验。

五、时间 & 空间复杂度

  • 时间复杂度：

    • get: O(1)
    • put: O(1)

  • 空间复杂度：O(capacity)

七、面试高频问题

  • Q: 为什么不用单向链表？
    A: 无法 O(1) 删除中间节点（不知道前驱）。
  • Q: 为什么 HashMap 不直接存 value，而要存 node？
    A: 需要通过 node 在链表中定位并移动/删除。
  • Q: 能否用 Java 自带的 LinkedHashMap​ 实现？
    A: 可以！重写 removeEldestEntry 即可，但面试通常要求手写。

class LRUCache {

    private class DoubleNode {
        private int key;
        private int val;
        private DoubleNode last;
        private DoubleNode next;

        private DoubleNode(int key, int val) {
            this.key = key;
            this.val = val;
        }

    }

    private class DoubleList {
        private DoubleNode head;
        private DoubleNode tail;

        private DoubleList() {
            head = null;
            tail = null;
        }

        private void addNode(DoubleNode newNode) {
            if (newNode == null) {
                return;
            }
            if (head == null) {
                head = newNode;
                tail = newNode;
            } else {
                tail.next = newNode;
                newNode.last = tail;
                tail = newNode;
            }
        }

        private void moveNodeToTail(DoubleNode node) {
            if (tail == node) {
                return;
            }
            if (head == node) {
                head = node.next;
                head.last = null;
            } else {
                node.last.next = node.next;
                node.next.last = node.last;
            }
            node.last = tail;
            node.next = null;
            tail.next = node;
            tail = node;
        }

        private DoubleNode removeHead() {
            if (head == null) {
                return null;
            }
            DoubleNode ans = head;
            if (head == tail) {
                head = null;
                tail = null;
            } else {
                head = ans.next;
                ans.next = null;
                head.last = null;
            }
            return ans;
        }

    }

    private Map<Integer, DoubleNode> keyNodeMap;

    private DoubleList nodeList;

    private final int capacity;

    public LRUCache(int capacity) {
        keyNodeMap = new HashMap<>();
        nodeList = new DoubleList();
        this.capacity = capacity;
    }

    public int get(int key) {
        if (keyNodeMap.containsKey(key)) {
            DoubleNode ans = keyNodeMap.get(key);
            nodeList.moveNodeToTail(ans);
            return ans.val;
        }
        return -1;
    }

    public void put(int key, int value) {
        if (keyNodeMap.containsKey(key)) {
            DoubleNode node = keyNodeMap.get(key);
            node.val = value;
            nodeList.moveNodeToTail(node);
        } else {
            if (keyNodeMap.size() == capacity) {
                keyNodeMap.remove(nodeList.removeHead().key);
            }
            DoubleNode newNode = new DoubleNode(key, value);
            keyNodeMap.put(key, newNode);
            nodeList.addNode(newNode);
        }
    }
}
